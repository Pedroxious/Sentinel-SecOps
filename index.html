Como Funciona

Pipeline automatizado de inteligência de ameaças

1
🔍

Monitoramento NVD

Varredura automatizada no National Vulnerability Database a cada 8 horas, capturando CVEs recentes com dados completos de CVSS, CWE e referências.

2
🧠

Análise com IA

Gemini 2.0 Flash analisa cada CVE gerando resumos executivos, avaliação de impacto e recomendações de mitigação — tudo em português.

3
🛡️

Inteligência de Ameaças

Cruzamento com CISA KEV (exploits ativos), FIRST EPSS (probabilidade), ThreatFox (IoCs) e GitHub Security Advisories (GHSA).

4
📊

Dashboard Interativo

Dashboard visual premium com filtros avançados, gráficos interativos, indicadores de risco e detalhamento completo de cada vulnerabilidade.

Fontes de Dados

APIs e feeds de inteligência integrados

NVD (NIST)

Base de dados oficial de CVEs do governo americano, com vetores CVSS e referências técnicas.

CISA KEV

Catálogo de vulnerabilidades conhecidas e ativamente exploradas, mantido pela CISA.

FIRST EPSS

Exploit Prediction Scoring System — probabilidade de exploração nos próximos 30 dias.

ThreatFox

Plataforma abuse.ch de Indicadores de Comprometimento (IoCs) — malware, C2, botnets.

GitHub Advisories

GitHub Security Advisories (GHSA) — vulnerabilidades em pacotes open-source e dependências.

Gemini AI

Análise semântica com Gemini 2.0 Flash — resumos, classificação de risco e recomendações em PT-BR.
